# GIT8

GitHub notification tool

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev -m electron
# or
yarn run dev
```

### Lint the files
```bash
yarn run lint
```

### Build the app for production
```bash
yarn run build
# or
quasar build
```
